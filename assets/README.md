# Assets 폴더

이 폴더에는 YOY Wallet App의 이미지 및 아이콘 파일들이 포함됩니다.

## 필요한 파일들

- `icon.png` - 앱 아이콘 (1024x1024)
- `splash.png` - 스플래시 스크린 이미지
- `adaptive-icon.png` - Android 적응형 아이콘
- `favicon.png` - 웹용 파비콘

## YOY 브랜드 가이드라인

- **주 색상**: 골드 (#FFD700)
- **보조 색상**: 블랙 (#000000)
- **로고**: YOY 심볼 사용
- **스타일**: 모던하고 미니멀한 디자인

## 이미지 크기 권장사항

- **앱 아이콘**: 1024x1024px
- **스플래시**: 1242x2436px (iPhone X 기준)
- **적응형 아이콘**: 1024x1024px
- **파비콘**: 32x32px
